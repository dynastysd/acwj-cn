# 翻译工作目录

本目录包含 ACWJ 项目中文翻译的管理文件。

## 文件说明

| 文件 | 说明 |
|-----|------|
| `README.md` | 本索引文件 |
| `GLOSSARY.md` | 翻译术语表 — 核心术语的中英文对照 |
| `PROGRESS.md` | 翻译进度追踪表 |
| `GUIDELINES.md` | 翻译规范与指南 |

## 快速开始

### 如何开始新的翻译任务

1. 查看 `PROGRESS.md` 选择待翻译章节
2. 查阅 `GLOSSARY.md` 确认术语翻译
3. 阅读 `GUIDELINES.md` 遵循翻译规范
4. 翻译对应的 Readme.md 文件
5. 更新 `PROGRESS.md` 的进度状态

### 术语表使用

在翻译过程中：

- **首次遇到术语** → 查 GLOSSARY.md
- **表中已有** → 直接使用
- **表中没有** → 自行决定翻译，记录并更新 GLOSSARY.md

### 批次翻译建议

由于 token 限制，建议每次翻译 1-3 章：

```
第1批: Part 0-2 (引言、扫描、解析)
第2批: Part 3-5 (优先级、编译器、语句)
第3批: Part 6-8 (变量、比较、if)
...
```
